Introduction

Goal Based Agents

Goal based agents are those intelligent agents in AI who focus on reaching the goal. Each of their actions is made towards the goal. Actually they compute at each state their cost to reach the goal and proceed optimally.
In AI the fields utilizing the goal based agents are travellling salesman, vlsi layout design, protien design, shortest route. All depends on the search agents and optimisation.

Representation

State Space

Each state in the state space is a node of the tree. A node, typically has some important information, which is parent, children, depth, path cost, and the associated state in the state space.

Algorithms

BFS

Breadth First Search is a simple uninformed searching algorithm that gives piority to the breadth before depth in a tree and the nodes covered first in a graph. It hence takes a queue for node storage to read the node entered first. It is generally used for shallow search depths when the goal state is basically in closer rows (as far as a tree is concerned).

DFS

Depth First Search is a different uninformed searching algorithm that takes into account the newly discovered nodes to be read first so that it could prioritise the depth of a tree over its breadth. To cover the newly discovered nodes we use stack as our node storage. Hence the stack constantly serves the newly entered nodes on a pop operation.

A*

A* is one of the best searching algorithms known till date. It is an informed search algorithm that takes into account the activation function for each node. Its activation function is the sum of path cost for the node and its heuristic value. It is implemented using a prioritize queue. This queue gets dynamically updated after each enqueue operation.
